# Exercise 01 · About Me

**Lesson:** 01 — What We're Building | **Difficulty:** Beginner

**Goal:** Create your own `about-me.md` file and push it to GitHub.

---

## Your Task

Create a file called `about-me.md` in the `jsbros-lab` repo. It must include all of the following:

- [ ] Your first name as a `#` heading
- [ ] One sentence about who you are
- [ ] Your top 3 favorite Roblox games as a list
- [ ] One thing you want to build (game, tool, or 3D print)
- [ ] One thing you're nervous or excited about learning

---

## Example

```md
# Jaxon

I'm 10 years old and I love Roblox and building stuff.

## My Top 3 Games
- Blox Fruits
- Brookhaven
- Adopt Me

## What I Want to Build
A Roblox game where you can collect coins and buy upgrades.

## One Thing I'm Excited About
Learning how to make things move with code.
```

---

## When You're Done

Run the four commands:

```bash
git status
git add .
git commit -m "add about-me"
git push
```

Then open GitHub and make sure your file shows up in the repo.

**You're done when:** Your `about-me.md` is visible on GitHub ✅
