# My Game Plan

## Project Name 
Coin Collecter 

## What Its Does
- Players run around a map **collecting** coins. 
- The player with the most coins after 60 seconds wins.

## Features

- A coin spawner that places 10 coins around the map 
- A score counter for each player A 60 second timer 
- A winner anoucement at the end

## How to Play 

1. Open the game in Roblox Studio and press Play 
2. Walk into yellow coins to collect them Watch your score go up 
3. See who wins when the timer runs out 

> Note this is a work in progress. Some features are not done yet.
